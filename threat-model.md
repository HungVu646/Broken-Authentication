# Threat Model - Cyber Fortress
## 1. Asset - Tài sản cần bảo vệ
- Tài khoản người dùng
- Dữ liệu cá nhân
- File hệ thống
- API key/token
- Tính sẵn sàng của dịch vụ
## 2. Threat - Mối đe dọa
- Người dùng độc hại
- Kẻ tấn công bên ngoài
- Bot/script tự động
- Người dùng bị lừa bấm link
## 3. Vulnerability - Lỗ hổng
- Mô tả lỗ hổng nhóm demo
## 4. Impact - Tác động
- Mất bí mật dữ liệu
- Mất toàn vẹn dữ liệu
- Mất tính sẵn sàng
- Chiếm quyền tài khoản
## 5. Mitigation - Biện pháp giảm thiểu
- Biện pháp kỹ thuật
- Biện pháp quy trình
- Biện pháp nâng cao nhận thức
